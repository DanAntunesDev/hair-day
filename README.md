# HairDay Booking API

API backend para gerenciamento de agendamentos.

## Tecnologias

- Node.js
- JavaScript
- Express
- REST API

## Funcionalidades

- Cadastro de usuários
- Criação de agendamentos
- Listagem de horários
- Atualização de dados

## Como rodar

```bash
git clone https://github.com/DanAntunesDev/hairday-booking-api

npm install

npm run dev
